# GagarinCatering

![GagarinCatering Preview](img/astronaut_1024.jpg)

Одностраничный сайт корпоративного кейтеринга с фронтендом на HTML/CSS/JavaScript и серверной обработкой на PHP.

Проект сочетает современный визуальный стиль, интерактивность и обработку заявок на стороне сервера.

## О проекте

GagarinCatering — лендинг компании по организации корпоративного кейтеринга.

Сайт предназначен для:

- Презентации услуг
- Приёма заявок на мероприятия
- Продвижения бренда
- Демонстрации форматов обслуживания

## Технологии

### Frontend

- HTML5
- CSS3 (Flexbox / позиционирование)
- JavaScript (ES6+)
- SVG-иконки
- Адаптивная верстка

### Backend

- PHP
- Обработка формы заявки
- Валидация данных
- Отправка email

## Функциональность

### Визуальная часть

- Hero-блок с крупным оффером
- CTA-кнопка «Заявка на мероприятие»
- Социальные ссылки
- Блок форматов мероприятий:
  - Банкет
  - Фуршет
  - Кофе-брейк
  - Открытая кухня

### Интерактивность (JavaScript)

- Открытие/закрытие формы
- Валидация полей перед отправкой
- Динамические эффекты (при наличии)
- Обработка событий пользователя

### Серверная логика (PHP)

- Приём данных из формы
- Проверка корректности
- Отправка письма администратору
- Защита от пустых/некорректных данных

## Валидация

- Клиентская валидация (JavaScript)
- Серверная валидация (PHP)
- Проверка обязательных полей
- Защита от отправки пустых данных

## Запуск проекта

1. Установить локальный сервер (OpenServer / XAMPP / MAMP)
2. Разместите файлы проекта в корневой директории веб-сервера
3. Запустить сервер
4. Открыть `index.html`
5. Backend на PHP не будет работать в системах похожих на Github
